# Hybrid Search KPI Dashboard

An end-to-end Hybrid Search system combined with a KPI Dashboard, built for a technical assessment.

## Features
- Hybrid Search API (BM25 + Vector Embeddings)
- Data Ingestion & Indexing
- Evaluation Harness (nDCG, Recall, MRR)
- KPI Dashboard & Metrics
- SQLite Logging

## Running the Application
To set up and run the system on a clean machine:
```bash
./up.sh
```
